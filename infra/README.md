# Infrastructure

## Purpose
Contains Infrastructure as Code (IaC) to deploy the Agentic Cloud Discovery platform on Azure.

## Tools
- **Bicep** (Preferred)
- Azure CLI

## Resources (MVP)
- Azure Container Apps (MCP Server, Orchestrator)
- Azure OpenAI
- Azure API Management
- Azure AI Search
- Azure AI Document Intelligence
- Azure Cosmos DB
- Azure Key Vault
- Application Insights + Log Analytics
