# Client UI

## Purpose
A minimal web UI to accept user queries and display the answer along with the execution trace.

## Features (MVP)
- Input field for user query.
- Display area for the final answer.
- Trace view showing the reasoning steps and tool executions.

## Tech Stack
- Minimal web framework (e.g., React, Streamlit, or simple HTML/JS).

See `docs/architecture/00-overview.md` for high-level details.
